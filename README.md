# FightReady (RingCall)

Muay Thai fight card management PWA. Fighters apply. Gyms approve. Stadiums match.

## Features
- **Fighter** — Browse events by region/stadium, apply to fight cards, track applications
- **Gym** — Approve fighters, manage roster, submit fighters to events
- **Stadium** — Create events, review eligible fighters, build fight cards
- Multi-language support (English, Thai, Chinese)
- Mobile-first responsive design with bottom nav
- Offline-capable PWA with service worker

## Run Locally
```bash
python -m http.server 3000
```
Then open [http://localhost:3000](http://localhost:3000)
